作业一：用户协议和注册页面

作业二：模仿商城页面布局


程序结构：

.
├── static # 静态资源目录
│   ├── css # css样式文件
│   ├── img   # 图片目录
│   └── js # javascipt脚本目录
├── index.html # 主页面
├── register.html # 注册页面
└── protocol.html # 用户协议页面

使用说明：进入主页面，点击右上角的注册链接，首先进入用户协议页面，用户点击同意并继续按钮后进入注册页面，填写注册信息后点击注册完成注册
